
# build and run

```bash
gradle clean
gradle build
gradle run
# or
java -cp app/build/classes/kotlin/main com.github.leventarican.salary.AppKt
```

# development setup

- Gradle 7.6
- Kotlin 1.7
- Java 17

# create project

```bash
mkdir kotlin-df
gradle init --type kotlin-application --project-dir kotlin-df
```
